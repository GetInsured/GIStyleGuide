Forms

Individual form controls receive styling, but without any required base class on the <form> or large changes in markup. In most cases, forms use the class.form-horizontal within the <form> tag.

When you have a label, it must be surrounded by a .control-group. For good accessibility practice, you should include a for tag that matches the input id.

<div class="control-group">
      <label class="control-label" for="inputEmail">Email</label>
      <div class="controls">
         <input type="text" id="inputEmail" placeholder="Email">
      </div>
   </div>

Modals

Modals are streamlined, but flexible, dialog prompts with the minimum required functionality and smart defaults.

Modal header

One fine body

Close Save changes 
<div class="modal hide fade">
   <div class="modal-header">
      <button type="button" class="close" data-dismiss="modal" aria-hidden="true"></button>
      <h3>Modal header</h3>
   </div>
   <div class="modal-body">
      <p>One fine body</p>
   </div>
   <div class="modal-footer">
      <a href="#" class="btn">Close</a>
      <a href="#" class="btn btn-primary">Save changes</a>
   </div>
</div>

Alerts & Notifications

Wrap any text and an optional dismiss button in .alert for a basic warning alert message.

Alerts

Alert Default

Add the class .alert for an alert.

Warning! Watch out for the kitty cat! 
<div class="alert">
   <button type="button" class="close" data-dismiss="alert"></button>
   <strong>Warning!</strong> Watch out for the kitty cat!
</div>

Alerts

Alert Success

Add the class .alert-success for a successful alert.

Well done! You found the kitty cat. 
<div class="alert alert-success">
   <button type="button" class="close" data-dismiss="alert"></button>
   <strong>Well done!</strong> You found the kitty cat.
</div>

Alerts

Alert Error

Add the class .alert-error for an error alert.

Oh snap! The kitty cat ran away! 
<div class="alert alert-error">
   <button type="button" class="close" data-dismiss="alert"></button>
   <strong>Oh snap!</strong> The kitty cat ran away!
</div>

Alerts

Alert Info

Add the class .alert-info for an info alert.

Heads up! The kitty cat needs food, but not too much because he's already fat. 
<div class="alert alert-info">
   <button type="button" class="close" data-dismiss="alert"></button>
   <strong>Heads up!</strong> The kitty cat needs food, but not too much because he's already fat.
</div>
