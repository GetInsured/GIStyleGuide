Iconograpahy

The icons GetInsured uses are very simple and to the point.

  • Icons are generally colorless with a blue background.
  • Be minimalistic when it comes to portraying your ideas.
  • Don't overcomplicate with too many details

    • Sprites

    With the number of images used on the GetInsured website, it is important to find ways to reduce the loading time to improve the responsiveness of the webpage.

    Sprites will allow us to reduce the number of HTTP requests that are made for each image and improve the overall performance. They do this by holding all the images used in one big master image, and then uses CSS and HTML to get the correct portions of the master image to display on the website. 

    Master Image:

    There are a many places you can use to generate a sprite master image along with the corresponding background position values. To generate the above example, we used CSS Sprites. Once the image is generated, use the background positions given to correctly position the elements in CSS/LESS. The resulting sprite is shown below (hover over it!). 

    Sprite:
    NOTE: The pie chart and the check mark are from the same master image

    The HTML for sprites is extremely simple. You can use <div>, <span>, <li>, and many others as long as you are changing the background image in your CSS/LESS.

    <div class="sprite coinsurance"> </div>

    The CSS/LESS looks complicated, but in reality it isn't that bad. Just a couple things to take note.

  • The position values are negative because you need to move the master image up and to the left to make sure you are correctly positioned on the right sprite.
  • Make sure you have the correct height and width or else you will end up displaying more/less than you want.
  • To get the sprite to look good, it will take a little tinkering with the values.

    • .sprite.coinsurance {
    background: url( ../images/sprite-example.png) -45px -65px;
    height: 70px;
    width: 70px;
}
.sprite:hover {
    background: url( ../images/sprite-example.png) 0px 10px no-repeat;
    height: 70px;
    width: 70px;
}

    Placeholder Images

    Placeholder images are exactly what they sound like. Use them when you don't have any images to display, but want someothing to hold their place.

    <div>
   <img class="placeholders" src="http://placehold.it/560x150">
</div>
<div>
   <img class="placeholders" src="http://placehold.it/350x250">
   <img class="placeholders" src="http://placehold.it/200x250">
</div>
    LOOK: These placeholders have their dimensions right on them! We got these from Placehold.it
    STOP: Be cautious when using these. They are responsive and will shrink if the window gets too small, so the dimensions, in some cases, might not be correct.
